Module 2 : DAX Time Intelligence (1.5h)

🟡 Intermédiaire

📚 Ce que vous allez apprendre

  • Créer une table DimDate complète
  • Calculer YTD, QTD, MTD
  • Comparer périodes (YoY, MoM)
  • Gérer les années fiscales
// Mesures Time Intelligence essentielles // 1. Year-to-Date Sales YTD = TOTALYTD( [Total Sales], Calendar[Date] ) // 2. Période précédente Sales LY = CALCULATE( [Total Sales], SAMEPERIODLASTYEAR(Calendar[Date]) ) // 3. Croissance Year-over-Year YoY Growth % = VAR _Current = [Total Sales] VAR _LastYear = [Sales LY] RETURN DIVIDE(_Current - _LastYear, _LastYear, 0) // 4. Derniers 12 mois (Rolling) Sales L12M = CALCULATE( [Total Sales], DATESINPERIOD( Calendar[Date], MAX(Calendar[Date]), -12, MONTH ) )

Module 4 : DAX Avancé (1.5h)

🟡 Intermédiaire

📚 Ce que vous allez apprendre

  • Maîtriser CALCULATE et modificateurs de contexte
  • Utiliser variables (VAR) pour lisibilité et performance
  • Créer des mesures avec SWITCH, IF imbriqués
  • Comprendre Filter Context vs Row Context
// Patterns DAX Intermédiaires // 1. CALCULATE avec ALL % of Total = DIVIDE( [Total Sales], CALCULATE( [Total Sales], ALL(Product) // Ignore filtres Product ) ) // 2. Variables complexes Customer Lifetime Value = VAR TotalRevenue = CALCULATE( SUM(Sales[Amount]), ALLEXCEPT(Sales, Sales[CustomerID]) ) VAR AvgMargin = 0.30 VAR AcquisitionCost = 50 RETURN (TotalRevenue * AvgMargin) - AcquisitionCost // 3. SWITCH pour catégorisation Customer Segment = VAR TotalSpent = [Total Sales by Customer] RETURN SWITCH( TRUE(), TotalSpent > 10000, "VIP", TotalSpent > 5000, "Gold", TotalSpent > 1000, "Silver", "Bronze" )
