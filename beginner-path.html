✏️ Exercice Pratique

🎯 Exercice 1.1 : Ma Première Table

Objectif : Créer une table des ventes par produit avec heatmap

Étapes :

  1. Insérer un visuel "Table"
  2. Ajouter ProductName et Sales[Amount]
  3. Activer conditional formatting > Background color > Gradient
  4. Trier par Sales décroissant
  5. Limiter à Top 10
✅ Résultat attendu : Table avec produits triés, couleurs du rouge (faible) au vert (fort)

Module 4 : Premières Mesures DAX (1h)

🟢 Débutant

📚 Ce que vous allez apprendre

  • Qu'est-ce qu'une mesure DAX ?
  • Formules de base : SUM, AVERAGE, COUNT
  • Calculs simples : différences, ratios
  • Fonction DIVIDE pour éviter erreurs
// Vos 5 premières mesures DAX // 1. Somme simple Total Sales = SUM(Sales[Amount]) // 2. Comptage Order Count = COUNTROWS(Sales) // 3. Moyenne Average Sales = AVERAGE(Sales[Amount]) // 4. Calcul (division sécurisée) Avg Order Value = DIVIDE( [Total Sales], [Order Count], 0 // Retourne 0 si division par zéro ) // 5. Pourcentage Margin % = DIVIDE( SUM(Sales[Amount]) - SUM(Sales[Cost]), SUM(Sales[Amount]), 0 )

✏️ Exercice Pratique

🎯 Exercice 4.1 : Créer 3 Mesures

Créez ces 3 mesures :

Puis affichez-les dans 3 Cards

Projet Final : Mon Premier Dashboard (1.5h)

🏆 Projet : Dashboard Ventes Complet

Créez un dashboard avec cette structure :

📊 ZONE SUPÉRIEURE (KPI)

5 Cards côte à côte :

  • Total Revenus
  • Total Profit
  • Marge %
  • Nombre Commandes
  • Panier Moyen

📈 ZONE CENTRALE (Analyse)

  • Bar Chart : Top 10 Produits
  • Line Chart : Ventes dans le temps

📋 ZONE INFÉRIEURE (Détails)

  • Table : Top 20 produits avec conditional formatting
💡 Checklist Qualité :
  • ✅ Tous les chiffres sont formatés (€, %, K/M)
  • ✅ Graphiques ont des titres clairs
  • ✅ Conditional formatting activé sur la table
  • ✅ Visuels alignés et espacés proprement
  • ✅ Couleurs cohérentes
